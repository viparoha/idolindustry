⚠️ 중요: 올바른 접속 방법 ⚠️

❌ 잘못된 방법:
- HTML 파일을 더블클릭
- 파일 탐색기에서 파일을 드래그해서 브라우저에 놓기
- 브라우저에서 File > Open으로 파일 열기

✅ 올바른 방법:
1. 브라우저를 엽니다 (Chrome, Safari, Firefox 등)
2. 주소창(URL 바)을 클릭합니다
3. 다음 주소를 직접 입력합니다:
   http://localhost:8000/beotulgi.html
4. Enter 키를 누릅니다

📝 단계별 설명:
1. 브라우저 주소창에 "localhost:8000" 입력
2. 자동완성이 나오면 선택하거나
3. 직접 "http://localhost:8000/beotulgi.html" 입력
4. Enter 키 누르기

🔍 확인 방법:
- 브라우저 주소창이 "http://localhost:8000/beotulgi.html"로 시작해야 합니다
- "file://"로 시작하면 안 됩니다!
